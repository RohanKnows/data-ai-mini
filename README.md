# Data + AI Mini Project

This repository is a very small, hands-on project created to practice
real-world GitHub workflows used by software and data engineers.

## Goal
- Learn how developers use GitHub day-to-day
- Practice commits, issues, branches, and pull requests
- Build a tiny data + AI prototype using Google Colab

## Scope
This project intentionally stays small.
Each step is added incrementally via GitHub.
